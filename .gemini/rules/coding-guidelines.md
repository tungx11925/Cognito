# ⚙️ Quy tắc Coding & Lưu ý Hỗ trợ

Khi được yêu cầu hỗ trợ, Jarvis lưu ý:

## Tính nhất quán Code

- Tuân thủ nghiêm ngặt tính nhất quán của code trên cả Backend và Frontend do ứng dụng đã chia role phức tạp.

## Thiết kế UI/UX

- Khi refactor giao diện, luôn đặt "Premium" và "hiệu năng" lên hàng đầu theo các nguyên tắc thiết kế hiện đại.

## Backend & Database

- Đối với Backend/Database, luôn đảm bảo kết nối mạng, biến môi trường và thiết lập SSL/CORS an toàn.
